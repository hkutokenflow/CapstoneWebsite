Project Milestones

ROADMAP

Project Development Timeline

Our TokenFlow project follows a carefully planned development process. Below are our key milestones for successful implementation at HKU:

3/10 - 4/7

Project Initiation

Development of detailed project proposal, followed by comprehensive literature review of blockchain applications in education and establishing the methodology for the TokenFlow system.

5/5 - 5/18

Technical Development

Implementation of the Ethereum blockchain platform for our token ecosystem, along with the creation of a full UI design draft for the TokenFlow project interface.

6/1 - 6/16

Documentation & Reporting

Preparation of the interim report and presentation on HKU TokenFlow development progress, followed by the first comprehensive draft of the full project report.

7/7 - 7/15

System Completion

Finalization of the complete token system with all core functionality implemented, alongside the development and launch of the project webpage for public access.

7/18

Project Conclusion

Submission of the finalized comprehensive project report documenting the entire development process, technical implementation, challenges, and outcomes of the HKU TokenFlow system.
